## Tech Stack

* A bunch of other cool stuff you can find in package.json

## Getting Started

1. Clone this bad boy:
   ```bash
   git clone <your-repository-url>
   cd contextcraft
   ```

2. Install dependencies:
   ```bash
   npm install
   ```
   (Grab a coffee, this might take a while...)

3. Run it:
   ```bash
   npm run dev
   ```
